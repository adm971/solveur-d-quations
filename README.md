# solveur equations
programme pédagogique pour générer et résoudre des équations du premier et second degré


## Prérequis

- Python 3.8+
- pip
- cx_Freeze

---

# Installation

## 1. Cloner le projet

git clone https://github.com/adm971/solveur-d-quations.git


---

# Installation sur Windows

1. Installer Python depuis :
https://www.python.org/downloads/

2. Installer cx_Freeze :

python -m pip install cx_Freeze

3. Compiler le programme :

python setup.py 

4. Lancer le programme :

un dossier "build" sera créé avec un sous dossier "exe.win" dans le quel se trouve "main.exe"
Lancer main.exe
